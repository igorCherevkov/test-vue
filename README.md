# 1. User guide

### 1. Просмотр сотрудников

На главной странице отображается таблица со списком сотрудников

- Таблица содержит следующие колонки:
- Используйте пагинацию внизу таблицы для навигации по страницам

### 2. Добавление нового сотрудника

- Нажмите кнопку "Добавить сотрудника" внизу страниц
- Заполните форму
- Нажмите кнопку "Сохранить"
- Новый сотрудник появится в таблице

### 3. Редактирование сотрудника

- Кликните по любой строке таблицы с сотрудником
- Откроется форма с текущими данными сотрудника
- Внесите необходимые изменения
- Нажмите кнопку "Сохранить"

# 2. Ключевые компоненты

- pages/index.vue - Главная страница с таблицей сотрудников
- components/EmployeeForm.vue - Компонент для отрисовки таблицы сотрудников
- components/EmployeeTable.vue - Компонент для отрисовки формы обновить / создать сотрудника
- data/employees.json - json с тестовыми сотрудниками
- server - mock запросы на серверы (просто обновляют json файл)

# 3. Запуск проекта

## Клонирование

```
git clone https://github.com/igorCherevkov/test-vue.git
cd test-vue
```

## Установка зависимостей

```
npm install
```

## Запуск приложения

- `npm run dev`: в режиме разработки,
- `npm run build`": сборка для продакшена,

## Доступ к приложению

- Frontend (Vue + Nuxt): http://localhost:3000
